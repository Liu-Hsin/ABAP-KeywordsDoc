ABAP-managed HDI container

AMHC for short. Special HDI container managed by AS ABAP. An AMHC is created and changed using transaction SCTS_AMHC. An AMHC and the contained HDI objects are connected to the Change and Transport System using SAP HANA Transport for ABAP (HTA). When an AMHC is created, a logical HDI container is defined and during its activation a physical HDI container is created in the SAP HANA database. The logical HDI container can be a logical schema in an AMDP macro where it represents the database schema of the physical HDI container.
