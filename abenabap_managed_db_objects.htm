ABAP-Managed Database Objects

By default, an AS ABAP operates with data stored in the ABAP database schema in its standard database. The schema can be accessed using a standard connection. The database objects used to store and access AS ABAP data in ABAP programs, that is, DDIC database tables, DDIC database views and stored procedures, are AS ABAP-managed objects. The database objects are not created and maintained natively in the development environment of the database platform itself. Instead, they are created and maintained using the associated AS ABAP tools and frameworks, such as the ABAP Dictionary, ABAP CDS, and AMDP.
