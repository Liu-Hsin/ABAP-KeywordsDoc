CDS DDL - DDIC-based View, path_expr, attributes

Syntax

... [ [1|*:] [INNER|{LEFT OUTER}] [WHERE] [cds_cond] ] ...

Effect

For every CDS association _assoc of a path expression of a CDS DDIC-based view, attributes for this part of the path expression can be defined in square brackets [ ]. The following attributes can be specified:

  • With 1|*:, the cardinality of the section can be specified.
  • With INNER|{LEFT OUTER} the join type can be specified.
  • With [WHERE] cds_cond a filter condition cds_cond can be specified.
