An addition, BASE, followed by an internal table, itab, can be specified in front of the specification of lines to be inserted, which is a functional operand position. The line type of itab must be convertible to the line type of the return value. If BASE is specified, the content of itab is assigned to the return value before the individual lines are inserted. If the character # is specified for the type of the return value and the type cannot be determined from the operand position of the constructor expression, the type of itab is used for this expression if it is known.
