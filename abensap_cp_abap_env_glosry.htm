SAP Cloud Platform, ABAP Environment

A special ABAP Platform variant. It is a part of the SAP Cloud Platform and is adapted to its requirements. SAP GUI cannot be used as a user interface. Any ABAP Workbench tools that are based on SAP GUI cannot be used. Application developers can use ABAP Development Tools (ADT) instead. The ABAP programming language can only be used in the ABAP language version ABAP for Cloud Development. ABAP programs, CDS entities, and ABAP Dictionary objects created by customers and partners can only access certain repository objects.
