ABAP CDS - SDL for Service Definitions

The CDS service definitions in ABAP CDS are created using the data definition language CDS SDL in SDL source code. CDS SDL complies with the general CDS syntax rules and contains the following language element for service definitions:

  • DEFINE SERVICE

Hint

The SDL source code of service definitions has its own editor in ADT, which is documented there. SDL source code can also be displayed in Repository Browser in ABAP Workbench.
