user master record

A user master record must exist that a user can log on to an AS ABAP. The record determines which actions a user with a certain user name is allowed to execute and which authorizations he or she possesses. Default settings, such as format settings for character strings are also recorded in the user master record. User master records are maintained in transaction SU01.
