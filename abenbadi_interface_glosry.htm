BAdI interface

A BAdI interface is a global interface that includes the tag interface IF_BADI_INTERFACE. A BAdI interface can be used as part of the definition of one or more BAdIs and may only contain methods (BAdI methods) and events but not variable attributes. Furthermore, a BAdI interface cannot contain any component interfaces other than other BAdI interfaces. An interface that includes a BAdI interface is also a BAdI interface.
