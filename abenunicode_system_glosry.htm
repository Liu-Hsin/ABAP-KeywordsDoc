Unicode system

Single code page system in which characters are encoded in a Unicode character representation. The system code page of a Unicode system is UTF-16 with platform-dependent byte order of which the ABAP programming language supports the subset covered by UCS-2. All ABAP programs in a Unicode system must have the ABAP language version Standard ABAP. In the current AS ABAP release, only Unicode systems are supported.
