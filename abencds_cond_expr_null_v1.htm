CDS DDL - DDIC-based View, cds_cond, NULL

Syntax

.... lhs IS [NOT] NULL ...

Effect

Identifies the null value in a CDS DDIC-based view. The relational expression is true if the value of lhs is (not) the null value.

The operand lhs must stick to the context-dependent rules listed in topic CDS DDL - DDIC-based View, cds_cond, Operands.

lhs can have any data type except of STRING, RAWSTRING, and GEOM_EWKB.
