dynpro - Screen and Screen Elements

The screen of a dynpro is created in Layout Editor in Screen Painter. The screen of the program DEMO_DYNPRO contains all possible screen elements. Above the screen, the menu bar, standard toolbar, title bar, and the application toolbar can be seen, with the status bar at the bottom.
