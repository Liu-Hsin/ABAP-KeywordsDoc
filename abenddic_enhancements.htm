DDIC - Enhancements

A DDIC structure, DDIC database table, DDIC database view, or domain in ABAP Dictionary created in a system can be enhanced in a different system using customizing includes, append structures, or DDIC append views without making any modifications. Customizing includes indicate points already marked by SAP for enhancements, whereas append structures can be appended anywhere.

  • Customizing includes
  • Append structures
  • DDIC append views
  • Fixed value appends

Search helps can be expanded using append search helps.

Enhancements are connected to Switch Framework whenever they are defined in a package that is assigned a switch.

Hint

For CDS views, CDS view extensions are an option for enhancements that is based on the CDS DDL of ABAP CDS.
