indicator structure

Substructure of a structured type. The indicator structure is the last component of the structured type and mirrors the preceding components as indicators of type x of length 1. Indicator structures can be defined with the addition WITH INDICATORS of the statement TYPES and can be used as ABAP SQL indicators.
