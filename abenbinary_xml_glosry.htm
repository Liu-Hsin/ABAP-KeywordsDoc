binary XML

SAP-specific format for the binary storage of XML data. In Binary XML, redundant information is suppressed, and UTF-8 encoding is used, which leads to performance benefits when storing and transporting data. In AS ABAP, Binary XML occurs as one of the supported formats in the sXML Library.
