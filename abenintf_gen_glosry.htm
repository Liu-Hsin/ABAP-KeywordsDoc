interface (general)

Boundary between two contexts. Procedures have a parameter interface. In classes, interfaces are defined by components of the different visibility sections. The public components of classes can be defined in special OO interfaces of ABAP Objects. The database interface connects an AS ABAP to its database. In special cases, interface work areas are used between programs of a program group The global data of the ABAP program or the interface work areas can be used as an interface between ABAP programs and dynpros.
