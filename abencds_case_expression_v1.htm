CDS DDL - DDIC-based View, case_expr

Syntax Forms

Simple case distinction

1. ... CASE operand
           WHEN operand1 THEN result1
          [WHEN operand2 THEN result2]
           ...
          [ELSE resultn]
      END ...

Complex case distinction

2. ... CASE WHEN cds_cond1 THEN result1
          [WHEN cds_cond2 THEN result2]
          [WHEN cds_cond3 THEN result3]
            ...
          [ELSE resultn]
      END ...



Effect

Case distinction in a SELECT statement of a CDS DDIC-based view. Either a simple case distinction (simple case) or a complex case distinction (searched case).

Case distinctions can be specified in the SELECT list and in operand positions of other expressions.
