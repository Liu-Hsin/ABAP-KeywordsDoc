streaming data sink

Repository in which a writer data stream stores its data in streaming. A streaming data sink can be an external resource such as a LOB column of a DDIC database table or a file, or an internal resource such as a string or an internal table. Streaming is currently implemented for DDIC database tables, strings, and internal tables.
