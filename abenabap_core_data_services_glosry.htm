ABAP Core Data Services

ABAP CDS for short. The ABAP Core Data Services implement the general CDS concept for AS ABAP. They use a CDS DDL to define CDS entities that implement a semantic data model. A CDS DCL makes it possible to develop CDS roles for CDS access control. CDS SDL and CDS BDL define CDS service definitions and CDS behavior definitions for ABAP RAP.
