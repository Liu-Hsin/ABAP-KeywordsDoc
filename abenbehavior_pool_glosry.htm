ABAP Behavior Pool

ABP for short. A class pool, in which the behavior of a RAP business object is implemented. A behavior pool is based on a CDS behavior definition. The global class of a behavior pool is introduced with BP_ but does not implement the behavior itself. The behavior implementation is coded in local classes in the CCIMP include of the behavior pool.
