text field

In ABAP, character-like data type for saving character strings of fixed length or a similar data object. The corresponding built-in ABAP type is c. The internal representation takes place in the character representation of the current code page.

In dynpros, a screen element that is used purely for displaying text and on which no user actions are possible. A text field of a dynpro is not linked to a dynpro field.
