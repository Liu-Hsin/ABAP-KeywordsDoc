CDS DDL - projection_view, element_list

Syntax

... proj_element1, proj_element2[,
    virtual_element1, virtual_element2, ... ]

Effect

Defines the elements of a CDS projection view. The elements are specified in a comma-separated list. The element list consists of projected elements (these can be fields, associations, compositions, or to-parent associations), and, optionally, virtual elements. There can be one or several of all of the listed elements.

The projected elements proj_element are the elements of the underlying CDS view view_entity or elements of the CDS associations exposed in the view_entity. Associations, compositions, and to-parent associations of the projected entity can be exposed as well. It is also possible to add new elements to a projection view which are not included in the projected entity. These elements are called virtual elements and they can be specified using the keyword VIRTUAL. For further details, see the respective sections below.

The names of the elements of a CDS projection view must be unique. They can be either the names of the elements taken from the data source, or else alternative element names specified using AS.
