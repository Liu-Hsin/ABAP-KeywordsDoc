code page

Mapping of a character set to bit sequences. For example, Latin1 (ISO-8859-1) represents a code page that contains all the characters required for Western European languages, whereas a Unicode character representation can represent all characters required world-wide. The way in which characters are represented in an internal session is determined by the code page of the text environment for the internal session. Each code page supported by SAP is assigned an SAP code page number in the DDIC database table TCP00. See also system code page.
