join expression

Implementation of a join in SQL using the addition JOIN. Join expressions are possible in Queries defined by the SELECT statement of ABAP SQL and the SELECT statement of ABAP CDS (SELECT in CDS view entities,SELECT in CDS DDIC-based views)
