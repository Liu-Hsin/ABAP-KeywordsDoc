ABAP RESTful Application Programming Model

Also ABAP RESTful Programming Model, ABAP RAP, or RAP for short. ABAP programming model that is RESTful or that meets the requirements of a REST architecture. In ABAP RAP, AS ABAP plays the role of a stateless Web server. OData serves as the RESTful protocol for requests from Web clients and the state of a session can be saved to the standard database. The modelling is based on CDS entities and special CDS RAP objects. For transactional access to data, the model offers RAP business objects, whose behavior is implemented in ABAP behavior implementations.
