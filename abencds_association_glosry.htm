CDS association

Association of ABAP CDS. CDS associations can be defined for CDS view entities, CDS DDIC-based views, CDS projection views, and CDS abstract entities using the addition ASSOCIATION in the CDS DDL in ABAP CDS. Associations define join relationships between association sources and association targets. They can be evaluated in CDS DDL, CDS DCL, and in ABAP SQL using SQL path expressions. CDS compositions and to-parent associations are special associations for the CDS composition tree of a RAP business object.
