ABAP behavior implementation

An ABAP Behavior implementation implements a RAP Business Object behavior. The behavior of a RAP business object is implemented in local classes of an ABAP behavior pool (ABP) as defined by a CDS behavior definition. There can be ABP handler and ABP saver classes. The local classes are called by the ABAP runtime environment, when a RAP BO is accessed.
