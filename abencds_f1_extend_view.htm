CDS DDL - EXTEND VIEW

The statement EXTEND VIEW in CDS DDL has two variants:

  • EXTEND VIEW ddic_based_view
This variant is used to define a CDS view extension that is also associated with a DDIC append view in ABAP Dictionary.
  • EXTEND VIEW ENTITY
This variant is used to define a CDS view extension that no longer needs a dictionary object.
