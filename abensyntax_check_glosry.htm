syntax check

Static check performed on ABAP source code to check whether the syntax is formally correct. The syntax check is called in the ABAP Editor via Program → Check → Syntax or via the statement SYNTAX-CHECK. An automatic syntax check that checks the source code while it is entered can be configured in the ABAP Development Tools (ADT).
