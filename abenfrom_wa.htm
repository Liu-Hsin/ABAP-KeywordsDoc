Effect

For wa, a work area compatible to the line type of the internal table must be specified. It is a functional operand positions. The first line found in the internal table, whose values in the columns of the table key used match those of the corresponding components of wa, is processed. If the key fields in wa are empty, no entries are processed.

If the USING KEY addition is not specified, the primary table key is used. If the USING KEY addition is specified, the table key specified in keyname is used.
