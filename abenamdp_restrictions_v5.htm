The following additional restrictions apply in the current ABAP language version ABAP for Cloud Development:

  • In declarations of the AMDP method using the addition AMDP OPTIONS READ-ONLY or in implementations of the AMDP method using the addition OPTIONS READ-ONLY, the AMDP procedure implementation must be restricted to reads.
  • The AMDP procedure must be implemented in SQLScript.
  • Only the repository objects released for the ABAP language version can be used.
  • In the implementation of the AMDP method, the name of the CDS entity must be specified after the USING addition for CDS views and CDS table functions. The name of a CDS-managed DDIC view or AMDP function is not possible.
  • Only objects in the current ABAP database schema can be accessed in the implementation. Other prefixes or corresponding AMDP macros are not allowed.
  • Any statements that could potentially cause problems are not allowed in the implementation. For example, no CE_... SQLScript operators are allowed and session variables cannot be accessed.
